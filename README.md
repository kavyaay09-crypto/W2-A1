# Task CRUD API

A simple REST API built using Node.js and Express.js that performs CRUD (Create, Read, Update, Delete) operations on tasks.

## Features

- Get all tasks
- Get a task by ID
- Create a new task
- Update a task
- Delete a task

## Technologies Used

- Node.js
- Express.js

## Installation

```bash
npm install
```

## Run the Project

```bash
node server.js
```

## API Endpoints

- GET /
- GET /health
- GET /tasks
- GET /tasks/:id
- POST /tasks
- PUT /tasks/:id
- DELETE /tasks/:id
